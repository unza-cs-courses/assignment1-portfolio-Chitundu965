About Me

Hello! I'm Chitundu Nakanyika, a fourth-year software engineering student at the University of Zambia with a deep passion for web development and software design. My journey into technology began with a curiosity about how websites work, which quickly evolved into a commitment to mastering both frontend and backend development.

Throughout my academic career, I've developed a strong foundation in computer science fundamentals, including data structures, algorithms, object-oriented programming, and database design. I'm currently enrolled in CSC4035 Web Programming and Technologies, where I'm honing my skills in modern web development practices using HTML5, CSS3, JavaScript, and responsive design principles.

What excites me most about software engineering is the intersection of logic and creativity. I believe the best applications are those that not only function flawlessly but also provide an intuitive and delightful user experience. This philosophy drives me to continuously improve my skills in both technical implementation and user interface design. Beyond coding, I'm passionate about graphic design and exploring how visual aesthetics can enhance digital experiences.
